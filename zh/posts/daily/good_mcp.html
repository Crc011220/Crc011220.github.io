Good MCP Server

Chrome DevTools MCP

Chrome DevTools MCP 是一个强大的浏览器自动化工具，它允许 AI 助手直接与 Chrome 浏览器进行交互。通过这个 MCP 服务器，AI 可以：

  • 网页自动化：自动点击、填写表单、导航页面等操作
  • 性能分析：分析网页性能指标，包括 Core Web Vitals
  • 调试支持：获取控制台消息、网络请求信息
  • 截图功能：捕获网页或特定元素的截图
  • 元素交互：悬停、拖拽、上传文件等操作

这个工具特别适合用于网页测试、自动化任务和性能监控。

Figma Developer MCP

Figma Developer MCP 提供了与 Figma 设计平台的深度集成，让 AI 助手能够：

  • 设计文件访问：读取和解析 Figma 设计文件的结构和内容
  • 设计资源下载：下载设计中的图片、图标等资源
  • 设计数据提取：获取设计规范、颜色、字体等信息
  • 设计协作：与设计团队协作，获取最新的设计更新

这个 MCP 服务器对于需要将设计转换为代码、提取设计规范或进行设计系统管理的开发团队非常有用。

Supabase MCP

Supabase MCP 是一个强大的数据库集成工具，它连接了 Supabase 的 PostgreSQL 数据库服务。通过这个 MCP，AI 可以：

  • 数据库操作：执行 SQL 查询、插入、更新、删除数据
  • 实时数据访问：访问 Supabase 的实时数据库功能
  • 身份验证集成：与 Supabase Auth 系统集成
  • API 管理：管理 Supabase 的自动生成的 API
  • 存储操作：访问 Supabase Storage 进行文件管理

这个工具特别适合需要数据库操作、用户管理和实时数据同步的应用开发。

Neon MCP

Neon MCP 提供了与 Neon 无服务器 PostgreSQL 数据库的连接。Neon 是一个现代化的数据库服务，通过 MCP 可以：

  • 无服务器数据库：访问 Neon 的无服务器 PostgreSQL 数据库
  • 数据库管理：创建、管理数据库实例和连接
  • 查询执行：执行复杂的 SQL 查询和数据分析
  • 性能监控：监控数据库性能和使用情况
  • 自动扩展：利用 Neon 的自动扩展功能

这个工具适合需要高性能、可扩展数据库服务的现代应用开发。

Vercel MCP

Vercel MCP 提供了与 Vercel 部署平台的集成，让 AI 助手能够：

  • 项目部署：自动化部署前端和后端项目到 Vercel
  • 域名管理：管理自定义域名和 DNS 设置
  • 环境变量：管理项目的环境变量配置
  • 性能监控：监控部署的性能和访问统计
  • 团队协作：管理团队权限和项目访问

这个工具特别适合需要自动化部署和监控的现代 Web 应用开发。

CloudFlare MCP

CloudFlare MCP 提供了与 CloudFlare 云服务的集成，包括：

  • CDN 管理：管理内容分发网络和缓存设置
  • DNS 管理：配置和管理 DNS 记录
  • 安全服务：配置防火墙、DDoS 防护等安全功能
  • 性能优化：优化网站性能和加载速度
  • SSL 证书：管理 SSL 证书和安全配置

这个工具适合需要全球 CDN、安全防护和性能优化的网站运维。

Stripe MCP

Stripe MCP 提供了与 Stripe 支付平台的集成，让 AI 助手能够：

  • 支付处理：处理信用卡、数字钱包等支付方式
  • 订阅管理：管理订阅服务和定期付款
  • 发票生成：自动生成和管理发票
  • 客户管理：管理客户信息和支付历史
  • 财务报告：生成支付和收入报告

这个工具特别适合需要集成支付功能的电商和 SaaS 应用开发。

Context7 MCP

Context7 MCP 是一个文档和知识库集成工具，它提供了：

  • 文档访问：访问各种技术文档和 API 文档
  • 代码示例：获取代码示例和最佳实践
  • 版本信息：获取软件包的版本和更新信息
  • API 文档：访问详细的 API 文档和参数说明
  • 社区资源：访问开源项目的社区资源

这个工具对于需要快速查找技术文档、API 信息和代码示例的开发者非常有用。

Github MCP

GitHub MCP 提供了与 GitHub 平台的全面集成，包括：

  • 仓库管理：创建、克隆、管理 GitHub 仓库
  • 代码操作：提交代码、创建分支、合并请求
  • Issue 管理：创建、更新、关闭 Issue 和 Pull Request
  • 团队协作：管理团队权限和项目访问
  • CI/CD 集成：与 GitHub Actions 集成进行自动化部署
  • 代码搜索：搜索代码库和用户信息

这个工具是软件开发团队进行版本控制、代码协作和项目管理的重要工具。
