# Good MCP Server

## Chrome DevTools MCP

Chrome DevTools MCP is a powerful browser automation tool that allows AI assistants to directly interact with Chrome browsers. Through this MCP server, AI can:

- **Web Automation**: Automatically click, fill forms, navigate pages, and perform other operations
- **Performance Analysis**: Analyze web performance metrics, including Core Web Vitals
- **Debugging Support**: Get console messages and network request information
- **Screenshot Functionality**: Capture screenshots of web pages or specific elements
- **Element Interaction**: Hover, drag, upload files, and other interactive operations

This tool is particularly suitable for web testing, automation tasks, and performance monitoring.

## Figma Developer MCP

Figma Developer MCP provides deep integration with the Figma design platform, enabling AI assistants to:

- **Design File Access**: Read and parse Figma design file structures and content
- **Design Resource Downloads**: Download images, icons, and other resources from designs
- **Design Data Extraction**: Extract design specifications, colors, fonts, and other information
- **Design Collaboration**: Collaborate with design teams and get the latest design updates

This MCP server is very useful for development teams that need to convert designs to code, extract design specifications, or manage design systems.

## Supabase MCP

Supabase MCP is a powerful database integration tool that connects to Supabase's PostgreSQL database service. Through this MCP, AI can:

- **Database Operations**: Execute SQL queries, insert, update, and delete data
- **Real-time Data Access**: Access Supabase's real-time database functionality
- **Authentication Integration**: Integrate with Supabase Auth system
- **API Management**: Manage Supabase's auto-generated APIs
- **Storage Operations**: Access Supabase Storage for file management

This tool is particularly suitable for application development that requires database operations, user management, and real-time data synchronization.

## Neon MCP

Neon MCP provides connection to Neon's serverless PostgreSQL database. Neon is a modern database service, and through MCP you can:

- **Serverless Database**: Access Neon's serverless PostgreSQL database
- **Database Management**: Create and manage database instances and connections
- **Query Execution**: Execute complex SQL queries and data analysis
- **Performance Monitoring**: Monitor database performance and usage
- **Auto-scaling**: Utilize Neon's auto-scaling capabilities

This tool is suitable for modern application development that requires high-performance, scalable database services.

## Vercel MCP

Vercel MCP provides integration with the Vercel deployment platform, enabling AI assistants to:

- **Project Deployment**: Automatically deploy frontend and backend projects to Vercel
- **Domain Management**: Manage custom domains and DNS settings
- **Environment Variables**: Manage project environment variable configurations
- **Performance Monitoring**: Monitor deployment performance and access statistics
- **Team Collaboration**: Manage team permissions and project access

This tool is particularly suitable for modern web application development that requires automated deployment and monitoring.

## CloudFlare MCP

CloudFlare MCP provides integration with CloudFlare cloud services, including:

- **CDN Management**: Manage content delivery networks and cache settings
- **DNS Management**: Configure and manage DNS records
- **Security Services**: Configure firewalls, DDoS protection, and other security features
- **Performance Optimization**: Optimize website performance and loading speed
- **SSL Certificates**: Manage SSL certificates and security configurations

This tool is suitable for website operations that require global CDN, security protection, and performance optimization.

## Stripe MCP

Stripe MCP provides integration with the Stripe payment platform, enabling AI assistants to:

- **Payment Processing**: Handle credit cards, digital wallets, and other payment methods
- **Subscription Management**: Manage subscription services and recurring payments
- **Invoice Generation**: Automatically generate and manage invoices
- **Customer Management**: Manage customer information and payment history
- **Financial Reports**: Generate payment and revenue reports

This tool is particularly suitable for e-commerce and SaaS application development that requires payment integration.

## Context7 MCP

Context7 MCP is a documentation and knowledge base integration tool that provides:

- **Document Access**: Access various technical documentation and API documentation
- **Code Examples**: Get code examples and best practices
- **Version Information**: Get package versions and update information
- **API Documentation**: Access detailed API documentation and parameter descriptions
- **Community Resources**: Access open source project community resources

This tool is very useful for developers who need to quickly find technical documentation, API information, and code examples.

## Github MCP

GitHub MCP provides comprehensive integration with the GitHub platform, including:

- **Repository Management**: Create, clone, and manage GitHub repositories
- **Code Operations**: Commit code, create branches, and merge requests
- **Issue Management**: Create, update, and close Issues and Pull Requests
- **Team Collaboration**: Manage team permissions and project access
- **CI/CD Integration**: Integrate with GitHub Actions for automated deployment
- **Code Search**: Search code repositories and user information

This tool is an essential tool for software development teams for version control, code collaboration, and project management.
